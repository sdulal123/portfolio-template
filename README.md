## Portfolio - 2024 (version 1)

## Tools

    - Next.js
    - Tailwind CSS
    - Framer Motion

## Build Instructions

```bash
# Install dependencies (only the first time)
npm install

# Run the local server at http://localhost:3000
npm run dev

# Build for production in the out/ directory
npm run build
```
